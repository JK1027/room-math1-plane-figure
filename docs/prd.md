# 프로젝트 개요

-   **프로젝트명:** Geometry Escape: 기하의 성
-   **프로젝트 목적:** 중학교 1학년 「평면도형의 성질」 단원을 게임
    기반으로 학습할 수 있는 GAS 기반 방탈출 게임 개발
-   **핵심 문제:** 기존 문제풀이 중심 수업의 흥미 부족
-   **타겟 사용자:** 중학교 1학년 학생(수업 및 가정학습)
-   **핵심 가치 제안:**
    -   스토리 기반 학습
    -   탐색 + 퍼즐 + 도형 조작
    -   교사용 학습 기록 자동 수집
    -   다른 단원으로 확장 가능한 게임 엔진

# 기술 설계

## 기술 스택

  영역       기술
  ---------- -------------------------
  Frontend   HTML5, CSS3, JavaScript
  Backend    Google Apps Script
  DB         Google Sheets
  그래픽     SVG + Canvas
  배포       GAS Web App

## 선택 이유

-   **GAS:** 무료, 학교 환경에 적합, 배포가 간단
-   **Google Sheets:** 별도 DB 없이 기록 관리 가능
-   **HTML/CSS/JS:** 유지보수가 쉽고 AI 협업에 적합
-   **SVG/Canvas:** 도형 조작 퍼즐 구현에 적합

### 대안

-   Firebase: 확장성은 높지만 관리 복잡도 증가
-   Supabase: 기능은 우수하지만 학교 환경에서는 과한 구성
